# Keen Keeper

Keen Keeper is a friendship interaction tracking app that helps you stay connected by logging calls, texts, and video chats in one simple place.

## Technologies Used

- Next.js
- React
- Tailwind CSS
- DaisyUI

## Key Features

1. Add and manage friend profiles.
2. Track interactions by type, including call, text, and video.
3. View interaction timeline and quick stats to understand your communication habits.
